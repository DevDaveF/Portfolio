#Portfolio Landing Page Example

This is a personal Portfolio landing page sample website.

Options to view the website:

A - Using Git 1 - Clone the repo on your local machine 2 - Open git repo in Visual Studio Code on your machine and use the "Live Preview" extension to view

B - Copy Code Manually from Git 1 - Create a new blank project in your visual studio code 2 - Copy the index.html code into a new blank html file in Visual Studio Code and save. 3 - Copy the CSS page into a new styles.css file in Visual Studio Code and save. 4 - Use the "Live Preview" extension in Visual Studio Code to view the webpage
